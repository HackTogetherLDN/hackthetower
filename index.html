Hack the Tower - London hackday

A hack day for developers a to come together and work on projects together, build new apps or just discover how others are using all the shiny technology out there. There are no rules or prizes to distract you, except the infinitely valuable prize of knowledge and experience.

At the start of the hackday we share any project ideas we have, or technology we want to discover. This allows everyone to get into groups and work together to create apps and learn new stuff.

Free Salesforce developer environment

Discover how to create mobile-first enterprise applications in the cloud with a free developer envrionment from Salesforce. Comprehensive docs and example source code is available to help you create quickly and effectively.
